# Intelligent Skill Analysis and Personalized Job Discovery System for Youth Employment

AI-powered job recommendation platform that analyzes resumes and recommends suitable jobs based on user skills and qualifications.

## Features
- Resume Upload
- Skill Analysis
- Personalized Job Recommendation
- AI-based Matching
- LinkedIn-style UI

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- NLTK

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Objective
To help students and unemployed youth discover suitable career opportunities using AI and Machine Learning technologies.
